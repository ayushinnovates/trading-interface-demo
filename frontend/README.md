# Bajaj Broking Frontend

React-based trading platform frontend with Material UI.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- Dashboard with statistics
- Instruments listing
- Order placement and management
- Trade history
- Portfolio view with P&L

## Technology Stack

- React 18
- TypeScript
- Material UI
- React Router
- Vite

## Development

The frontend runs on `http://localhost:3000` and proxies API requests to `http://localhost:3001`.
